# Fractal Medical Analytics

Веборієнтований застосунок для оброблення медичних часових рядів з фрактальною структурою, виділення інформативних ознак сигналів та їх використання в задачах інтелектуального аналізу даних.


---

## Про проект

Система виконує повний цикл роботи з медичними часовими рядами:

- Імпорт даних у форматі CSV / XLSX
- Автоматичне визначення структури колонок
- Попереднє оброблення (нормалізація, усунення викидів, заповнення пропусків)
- Сегментація сигналу
- Обчислення фрактальних та статистичних ознак
- Інтелектуальний аналіз: класифікація, кластеризація, пошук аномалій
- Візуалізація результатів
- Формування PDF-звіту

---

## Фрактальні характеристики

| Метод | Опис |
|---|---|
| Показник Херста (H) | Довготривала залежність та самоподібність сигналу |
| DFA-експонента (α) | Масштабна поведінка флуктуацій часового ряду |
| Фрактальна розмірність Хігучі | Складність графіка сигналу на різних кроках |
| Фрактальна розмірність Каца | Геометрична складність форми сегмента |
| Фрактальна розмірність Петросяна | Нерегулярність сигналу через підрахунок змін знаку |

## Технології

- **Python 3.11+**
- **Django** — веб-фреймворк
- **SQLite** — база даних
- **scikit-learn** — алгоритми машинного навчання

## Запуск

```bash
# 1. Створити віртуальне середовище
python -m venv .venv

# 2. Активувати (Windows)
.venv\Scripts\activate

# 3. Встановити залежності
pip install -r requirements.txt

# 4. Застосувати міграції
python manage.py migrate

# 5. Завантажити демо-дані
python manage.py seed_demo

# 6. Запустити сервер
python manage.py runserver
```

Відкрити у браузері: **http://127.0.0.1:8000**

**Логін:** `demo`  
**Пароль:** `demo12345`

## Режими аналізу

- **Класифікація** — Random Forest, SVM, kNN (потребує мітки класів)
- **Кластеризація** — K-means, Agglomerative (без міток)
- **Пошук аномалій** — Isolation Forest (виявлення нетипових сегментів)

---

## Формат вхідних даних

Файл CSV або XLSX з числовими колонками. Мінімальна структура:

```
time, value
0.0, 1.23
0.01, 1.45
...
```

Підтримуються також колонки `label` та `subject_id`.

##  Застереження

Програмний засіб призначений виключно для дослідницького та демонстраційного аналізу медичних часових рядів. Він не є медичним діагностичним інструментом і не може використовуватися для встановлення діагнозу.
